# prismarine-physics

Accurate physics for Minecraft 1.8.9. Tested against vanilla client movements and Grim AC.

---

## Installation

```
npm install github:binmasterdotpro/prismarine-physics#master
```

## Limitations

 * Only meant for Minecraft 1.8.9
 * Some features not yet implemented (ie. elytra, rideable entities, flying, etc.)
